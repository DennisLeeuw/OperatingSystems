Debian gebruikt voor het beheren van IP-netwerken bestanden in de \texttt{/etc/network} directory. Een uitzondering is het \texttt{networking} bestand in de \texttt{/etc/default} directory. In dit bestand kan je een variabele zetten die voorkomt dat het netwerk geconfigureerd wordt. Dit laatste bestand zullen we hier niet behandelen. Wat we ook niet behandelen is WiFi. De meeste server-systemen zullen geen WiFi gebruiken omdat het te onbetrouwbaar is. Wij gaan kijken hoe we een ethernet-netwerk van IP kunnen voorzien.
