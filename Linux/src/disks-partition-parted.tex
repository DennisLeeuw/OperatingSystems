De belangrijkste feature van \texttt{parted} is de mogelijkheid om een partitie te resizen (vergroten en verkleinen). De tool kent een commandline modus en een interactieve modus.

Een syntax overzicht van hoe je \texttt{parted} kan gebruiken op de commandline:
\begin{lstlisting}[language=bash]
sudo parted mkpart part-type name fs-type start end
\end{lstlisting}
