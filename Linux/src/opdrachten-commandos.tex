\begin{enumerate}
\item In mijn home-directory wil ik een directory maken met de naam \texttt{Cursus}. In deze directory moeten twee directories komen. De namen van die directories zijn \texttt{Shell} en \texttt{Commandos}. Pas als de directory \texttt{Cursus} is aangemaakt zonder fouten wil ik dat de overige twee directories worden aangemaakt. De opdracht moet \'e\'en opdrachtregel zijn. Hoe ziet de opdrachtregel eruit?
\item Het commando \texttt{file} laat zien wat voor bestandstype een bestand is. Het gebruikt een bestandsnaam als argument. Hoe zorg je ervoor dat je van alle bestanden in je home-directory te zien krijgt wat voor bestandstype het is?
\end{enumerate}
