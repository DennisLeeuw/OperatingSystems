Na het bestuderen van dit document heeft de lezer kennis van:
\begin{itemize}
\item het opvragen van IP-adressen op een GNU/Linux systeem
\item het opvragen van de route-tabel op een GNU/Linux systeem
\item het op een simpele manier testen van domainname servers
\item het opvragen van actieve TCP/IP poorten op een GNU/Linux systeem
\item waar de configratie bestanden op een GNU/Linux systeem staan voor het configureren van ethernet en WiFi
\item waar de configratie bestanden op een GNU/Linux systeem staan voor het configureren van TCP/IP IP-adressen en DHCP
\item waar de configratie bestanden op een GNU/Linux systeem staan voor het configureren van TCP/IP zetten van de default gateway
\item waar de configratie bestanden op een GNU/Linux systeem staan voor het configureren van TCP/IP zetten van de domainname servers
\end{itemize}
