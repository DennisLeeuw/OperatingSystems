Een configratie bestand dat per distributie anders is is niet makkelijk. Er is een gezamenlijke afspraak gemaakt dat er een identiek bestand is op elk systeem met de naam \texttt{os-release}, dit bestand bevat meer informatie dan alleen het versie-nummer:
\begin{lstlisting}[language=bash]
$ cat /etc/os-release
\end{lstlisting}
