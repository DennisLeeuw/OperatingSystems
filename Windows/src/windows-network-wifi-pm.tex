De power management features van Windows 11 kunnen soms in de weg zitten bij sommige netwerk adapters. Om power management op de WiFi-adapter uit te schakelen doe je het volgende:
\begin{enumerate}
\item Open Computer Management en selecteer Device Manager.
\item Bij Network Adapters selecteer je je WiFi adapter en klikrechts om Properties te kunnen selecteren.
\item Ga naar de Power Management tab en uncheck \textquote{Allow the computer to turn off this device to save power}.
\end{enumerate}

Kijk of dit het probleem met de WiFi adapter verhelpt. Als dat niet zo is kan je de setting weer terug zetten.
