De problemen voor bekabelde netwerken bestaan voornamelijk uit het niet verbonden zijn met een kabel of het niet verkregen hebben van een IP adres van een DHCP-server. Andere problemen kunnen te maken hebben meteen verkeerde default gateway of de verkeerde DNS servers. Als je al deze settings na gelopen hebt en je weet zeker dat ze goed zijn controlleer dan met:
\begin{description}
\item[ipconfig] welk IP adres je hebt en wat je DNS servers zijn en wat je default gateway.
\item[ping] tot waar je een netwerk verbinding hebt (ping je eigen IP, ping de default gateway, ping een IP-adres op Internet (bijvoorbeeld 8.8.8.8))
\item[nslookup] een domein naam en zie of je een IP adres terug krijgt
\end{description}
