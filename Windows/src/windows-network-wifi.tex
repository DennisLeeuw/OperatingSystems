Bij het gebruik van een draadloos netwerk zoals WiFi kunnen er vele problemen optreden, varierent van verkeerde wachtwoorden tot het niet hebben van voldoende signaal. In deze sectie gaan we kijken naar problemen die met Windows te maken kunnen hebben. Het is belangrijk om eerste andere zaken te controleren, zoals voldoende signaal sterkte voordat je de volgende stappen gaat doorlopen. E\'en van de zaken die problemen kunnen veroorzaken is het feit dat access points een bepaald maximum aan MAC-addressen kunnen hebben. Is dit maximum bereikt dat kan er daarna geen extra werkstation meer gebruik maken van dat access point. Dit probleem is vanaf een werkstation niet zichtbaar en kan voor eindeloos troubleshooting zorgen.
